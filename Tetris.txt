Tetris Game Main Design:
Game Board:
 - This will presumably be an array with 0 representing empty space and 1 representing occupied space

Player:
 - Maintain stats like high score, username, or things along that nature

Tetris Game (During Play):
 - This should maintain the current score, the current elapsed time.
 - This should allow pausing either by selecting a button to the side or pressing esc
 - As per the rules of Tetris, this should also have the space to store a piece, and allow the user to see the next 3 pieces
 - This should also have a game speed that slowly speeds up to a max speed like the Tetris game does.
 - Should have sound effects for placing pieces, showing tick speed (potentially), and clearing a row

Tetris Game (Outside Game):
 - Should show the top 5 scores of all time with 1 section for players and another for the best AI model
 - Should allow the user to select which game mode to play

Hierarchy:

GameBoard.py:
	This file contains the gameBoard and the current state of the game.
	This will be responsible for everything relating to or interacting with the board.
	
Piece.py:
	This file will contain the different pieces in the game.
	This will also do everything relating to the pieces like rotations.
